## ü©∫ Self-Diagnosis

Run the built-in diagnostic tool:

```bash
gev doctor
```

This will check:
- Node.js version compatibility
- SOPS installation and version
- age installation and version
- Git repository status
- Configuration file validity
- Policy file integrity

## üêõ Found a Bug?

Please [open an issue](https://github.com/pas7-studio/git-env-vault/issues/new?template=bug_report.yml) with:

1. Your environment details (OS, Node.js, SOPS, age versions)
2. Steps to reproduce the issue
3. Expected vs actual behavior
4. Any relevant logs (without secret values!)

**Important:** Never include secret values or private keys in issue reports.
